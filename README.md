# Databases — KBTU Fall 2025

Course: Databases
University: Kazakh-British Technical University (KBTU)

## Course Overview

Fundamentals of database design and management. Covers relational database theory,
SQL querying, normalization, transactions, and practical work with real database systems.

## Topics Covered

- Relational database model
- Entity-Relationship (ER) diagrams
- SQL: DDL, DML, DCL
- SELECT queries, JOINs, subqueries
- Aggregation and grouping
- Indexes and optimization
- Normalization (1NF, 2NF, 3NF, BCNF)
- Transactions and ACID properties
- Stored procedures and triggers
- PostgreSQL and MySQL

## Tech Stack

- SQL (PostgreSQL, MySQL, SQLite)
- DataGrip
- ER diagram tools
